# ISBN Validator (Python)

A Python program to validate ISBN-10 and ISBN-13 codes.

## Features
- Validates ISBN-10 and ISBN-13
- Handles invalid input safely
- Detects wrong length and invalid characters

## Example
Input:
1530051126,10

Output:
Valid ISBN Code.

## Concepts used
- Functions
- List comprehension
- Exception handling
- String slicing

Built while learning Python fundamentals.
